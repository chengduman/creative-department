# 第一部分：部门制度

## 一、部门定位
创意部是公司的视觉与内容呈现中心，负责PPT制作、信息图设计、可视化创作、文案策划，以及**报告排版与PDF终版制作**，确保公司研究成果以专业、美观的方式呈现。

## 二、工作原则
1. **专业美学** —— 保持高端咨询机构的视觉调性
2. **内容为王** —— 形式服务于内容，不炫技不堆砌
3. **快速交付** —— 标准化模板支撑高效产出
4. **统一品牌** —— 保持中普咨询视觉识别的一致性

## 三、工作流程

### 3.1 设计需求流程
1. 接收设计需求（调研报告配图/PPT/信息图/排版）
2. 明确用途、受众、风格要求
3. 设计方案 → 部长确认
4. 执行制作 → 内部审核 → 交付

### 3.2 报告排版流程（新增）
1. 接收Markdown源文件和排版标准
2. 检查排版规范（字体/字号/表格/图片/封面）
3. 执行PDF生成（Pandoc+Typst流水线）
4. 逐页检查排版质量
5. 交付终版PDF + 归档源文件

### 3.3 创意资产类型
| 类型 | 说明 |
|------|------|
| 报告插图 | 调研报告中的示意图、流程图 |
| 报告排版 | 调研报告PDF终版制作（Pandoc+Typst流水线） |
| 信息图（Infographic） | 数据可视化长图、一图读懂 |
| PPT演示 | 客户提案、内部汇报、行业分享 |
| 架构图 | 组织架构、产业链图谱、生态图谱 |
| 视频/动画 | 研究短视频、科普动画（Manim/p5js） |

## 四、品牌视觉规范
- 主色调：深蓝（#1a365d）+ 金色（#d69e2e）
- 字体：中文使用思源黑体，英文使用 Inter
- Logo：中普咨询 ZP Consulting
